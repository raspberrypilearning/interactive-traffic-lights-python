## Connecter les composants

\--- task \---

- Raspberry Pi
- 3x LEDs et résistances
- Bouton
- Buzzer
- Plaquette d'expérimentation
- Fils de connexion

Branche tes composants aux broches GPIO suivantes :

| Composant  | Broche GPIO |
| ---------- |:-----------:|
| Bouton     |     21      |
| LED rouge  |     25      |
| LED orange |     28      |
| LED verte  |     27      |
| Buzzer     |     15      |
