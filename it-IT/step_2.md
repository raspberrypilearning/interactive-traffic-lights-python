## Connect the components

\--- task \---

- Raspberry Pi
- 3x LED e resistenze
- Pulsante
- Cicalino
- Breadboard
- Ponticelli

Wire up your components to the following GPIO pins:

| Componente | Pin GPIO |
| ---------- |:--------:|
| Pulsante   |    21    |
| LED rosso  |    25    |
| LED ambra  |    28    |
| LED verde  |    27    |
| Cicalino   |    15    |
