## Di cosa avrai bisogno

### Hardware

- Raspberry Pi
- 3x LED e resistori
- Pulsante
- Cicalino
- breadboard
- Fili di ponticello

Collega i tuoi componenti ai seguenti pin GPIO:

| Componente | Pin GPIO |
| ---------- |:--------:|
| Pulsante   |    25    |
| LED rosso  |    24    |
| LED ambra  |    23    |
| LED verde  |    22    |
| Cicalino   |    5     |
