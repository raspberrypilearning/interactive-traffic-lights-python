## Connect the components

\--- task \---

- Raspberry Pi
- 3 LED's en weerstanden
- Drukknop
- Zoemer
- Experimenteer bord
- Verbindingsdraden

Wire up your components to the following GPIO pins:

| Component  | GPIO-pin |
| ---------- |:--------:|
| Drukknop   |    21    |
| Rode LED   |    25    |
| Oranje LED |    28    |
| Groene LED |    27    |
| Zoemer     |    15    |
