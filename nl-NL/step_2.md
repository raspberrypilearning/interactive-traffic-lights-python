## Wat heb je nodig

### Hardware

- Raspberry Pi
- 3 LED's en weerstanden
- Drukknop
- Zoemer
- Experimenteer bord
- Verbindingsdraden

Sluit uw componenten aan op de volgende GPIO-pinnen:

| Component  | GPIO-pin |
| ---------- |:--------:|
| Drukknop   |    25    |
| Rode LED   |    24    |
| Oranje LED |    23    |
| Groene LED |    22    |
| Zoemer     |    5     |
