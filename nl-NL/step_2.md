## Verbind de componenten

\--- task \---

- Raspberry Pi
- 3 LED's en weerstanden
- Drukknop
- Zoemer
- Experimenteer bord
- Verbindingsdraden

Sluit uw componenten aan op de volgende GPIO-pinnen:

| Component  | GPIO-pin |
| ---------- |:--------:|
| Drukknop   |    21    |
| Rode LED   |    25    |
| Oranje LED |    28    |
| Groene LED |    27    |
| Zoemer     |    15    |
