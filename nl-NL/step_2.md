## Wat heb je nodig

### Hardware

- Raspberry Pi
- 3 LED's en weerstanden
- Schakelaar
- Zoemer
- Experimenteer bord
- Verbindingsdraden

Sluit je componenten op de volgende GPIO-pinnen aan:

| Onderdeel  | GPIO-pin |
| ---------- |:--------:|
| Schakelaar |    25    |
| Rode LED   |    24    |
| Oranje LED |    23    |
| Groene LED |    22    |
| Zoemer     |    5     |
