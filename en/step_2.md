## What you will need

### Hardware

- Raspberry Pi
- 3x LEDs and resistors
- Button
- Buzzer
- Breadboard
- Jumper wires

Wire up your components to the following GPIO pins:

| Component | GPIO pin |
| --------- | :------: |
| Button    | 25       |
| Red LED   | 24       |
| Amber LED | 23       |
| Green LED | 22       |
| Buzzer    | 5        |
