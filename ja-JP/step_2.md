## あなたが必要とするもの

\--- task \---

- ラズベリーパイ
- 3x LEDおよび抵抗
- ボタン
- ブザー
- ブレッドボード
- ジャンパ線

Wire up your components to the following GPIO pins:

| 成分      | GPIOピン |
| ------- |:------:|
| ボタン     |   24   |
| 赤色LED   |   25   |
| 琥珀LED   |   23   |
| グリーンLED |   22   |
| ブザー     |   5    |
